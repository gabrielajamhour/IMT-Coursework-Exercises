# IMT Coursework Exercises

This repository contains small coding exercises developed during my time at the Instituto Mauá de Tecnologia (IMT) in Brazil.

## 📌 Context

After graduating from high school in December, I had a gap period before starting my university studies in Spain in September. During these six months, I enrolled at IMT to strengthen my foundation in programming and gain early exposure to university-level coursework.

## 🎯 Purpose

The goal of this repository is to document that initial learning phase, where I built core programming skills and developed a problem-solving mindset.

## 📚 Content

The repository includes:

- Introductory programming exercises  
- Basic algorithmic problems  
- Practice assignments from coursework  

## 🛠️ Technologies

- Python

## 📈 Note

These exercises represent an early stage of my learning journey. While simple, they reflect the foundation upon which I have continued to build more advanced technical skills.
